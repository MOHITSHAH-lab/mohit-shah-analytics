# Inventory Management Analytics Framework

## Project Overview

This project designs an **end-to-end Inventory Management Analytics Framework** that helps organizations improve inventory visibility, reduce working capital lock-in, and prevent revenue loss caused by stockouts.

The solution combines **inventory KPIs, demand variability analysis, supplier performance metrics, and reorder optimization logic**.

---

## Business Problems Solved

• Overstock leading to working capital lock-in
• Revenue loss due to stockouts
• Poor supplier and distributor performance visibility
• Inefficient reorder planning
• Lack of SKU prioritization

---

## Analytical Methods Used

* ABC Inventory Classification
* XYZ Demand Variability Classification
* FSN Movement Classification
* EOQ (Economic Order Quantity)
* Coefficient of Variation (CV)
* Pareto Analysis (80/20 Rule)
* GMROII Profitability Analysis

---

## Dashboard Components

The analytics dashboard includes:

• Inventory health KPIs
• Demand vs supply trends
• Stockout impact analysis
• Reorder point monitoring
• Supplier performance metrics
