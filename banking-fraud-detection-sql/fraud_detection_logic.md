Rule 1: Rapid Transaction Reversal
If a transaction is reversed within 24 hours, flag the account.

Rule 2: Dormant Account Activity
If an account inactive for more than 180 days suddenly performs transactions, flag the account.

Rule 3: Structuring Deposits
Multiple deposits below reporting threshold indicate possible regulatory evasion.

Rule 4: Circular Transactions
Funds moving through multiple accounts and returning to origin indicate layering.
